## Setup environment

```
1. Memasang streamlit virtual environment di folder project. Pada project ini kita akan memasang dan menggunakan venv. Perintah yang digunakan untuk install venv di cmd, kita dapat menjalankan perintah **python -m pip install venv**

2. Masih di cmd. Setelah virtual environment terpasang, langkah berikutnya membuat virtual environment baru pada project dengan perintah **python -m venv myprojectenv**

3. Setelah itu menjalankan "activate.bat". Karena saya dapat kendala di sini, saya menggunakan alamat directorinya langsung seperti ini **"D:\GitHub\Dicoding_Learn-Data-Scientist\Submission Project Analisis Data\myprojectenv\Scripts\activate.bat"**

4. Setelah masuk di virtual environment yang telah dibuat,  lakukan instalasi library yang dibutuhkan dan jangan lupa juga membuat requirements.txt yang isinya library yang dibutuhkan. Untuk tahap ini dapat menggunakan perintah **pip install -r requirements.txt**
```

## Run steamlit app

```
5. Selanjutnya, run streamlit dengan perintah <strong>streamlit run submission_project_analisis_data.py<strong>
```
